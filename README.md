# Менеджер задач (Task Manager)

## Описание проекта

Данное консольное приложение разработано на языке Python как итоговый проект в рамках обучения в «Малой академии программистов». Цель проекта — демонстрация навыков работы с логикой Python, структурой кода и пользовательским вводом.

## Функционал

Программа представляет собой инструмент для управления списком дел, включающий следующие возможности:

* Просмотр текущего списка задач.
* Добавление новых записей.
* Удаление задач из списка.
* Редактирование статуса (отметка «выполнено»).
* Полная очистка списка задач.
* Автоматическое сохранение и загрузка данных из файла `tasks.json`.

## Инструкция по запуску

1. Убедитесь, что на вашем компьютере установлен Python.
2. Скачайте файлы проекта в одну директорию.
3. Откройте командную строку или терминал в папке с проектом.
4. Запустите приложение командой:
